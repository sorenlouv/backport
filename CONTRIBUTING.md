# Contributing

### Run

```
yarn start --branch 6.1 --upstream sqren/backport-demo --all
```

or

```
# Compile
yarn tsc

# Run
node dist/index.js --branch 6.1 --upstream sqren/backport-demo --all
```

### Debug

**Run tests**

```
yarn test
```

**Run tests continously**

```
yarn test --watch
```

**Compile typescript continously**

```
yarn tsc --watch
```
